### Допущения
- унарного плюса нет
- 0**0=1
- все операции разделены пробелом при вводе
- унарный минус (~)  - операция
- обозначения операций: '//' - целочисленнрое деление,'**' - возведение в степень, '%' - остаток от деления,'~' - унарный минус, '|' - модуль
- округление до двух знаков после запятой (поменять можно через константы ROUNDING_DIGITS)
- .5 допускается и считается как 0.5
- //, % применяются только для целых чисел

### Цель проекта
Реализация на языке Python RPN-калькулятора, который принимает от пользователя выражения в формате обратной польской записи, выполняет их корректное вычисление с использованием стека и выводит результат.

### Библиотеки
re, pytest
### Функционал
- calculator_rpn() - реализация калькулятора
- correct_bracket() - проверка корректности выражения в скобках
- correct_enter() - проверка корректного ввода
- division() - деление с учётом обработки ошибок
- int_division() - целочисленное деление с учетом обработки ошибок
- percent() - остаток от деления с учётом обработки ошибок
- pow() - возведение в степень
### Запуск
python main.py
